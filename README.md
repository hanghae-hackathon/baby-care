## 💬 프로젝트 소개

이 프로젝트는 All-in-One 베이비시터 중개 웹서비스 입니다. 60세 이상의 노인 대상 베이비시터 교육과정 지원으로 노인 일자리 창출을, 정부 지원을 통해 아이 양육 부담을 줄일 수 있습니다.
